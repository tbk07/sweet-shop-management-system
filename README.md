# AI Kata Sweet Shop Management System

## Overview
A full-stack management system for sweet shops supporting product/inventory management, customer orders, and reporting.

## Features
- Customer authentication, product browsing, purchase by quantity
- Admin inventory and product management
- Advanced reporting and analytics
- Responsive React frontend & Express backend

## Tech Stack
- Express (Node.js), REST API, JWT
- React 18, Bootstrap 5, Axios

## Installation

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd client
npm install
npm start
```

## Usage
- Register/login as a customer or admin.
- Browse & buy sweets.
- Admin: manage products, view reports.

## API Endpoints
- `/api/products` - List sweets
- `/api/orders` - Place order
- `/api/customers` - Customer management

## Testing

### Frontend (Jest)

### Backend (API Test Example)
